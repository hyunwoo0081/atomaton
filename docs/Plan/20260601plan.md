# 발견한 버그 목록

- [x] 워크플로우를 변경할 때 마다, 웹 훅의 URL 이 변경됨.
  - [x] 워크플로우를 변경해도, 웹훅 URL 과, 인증정보는 변하면 안됨.
- [x] Discord Webhook 을 연결했을 때 변수가 적용되지 않음. Trigger 또는 앞에서 나온 변수에 대한 적절한 변수를 선택할 수 있도록 제공해야 하며, 변수가 실제로 메세지에 반영되어야 함.
- [x] 워크플로우에서 Save 를 눌렀을 때, mutation 시 pendding 을 넣고, 같은 내용을 저장하지 않도록 UI를 개선해야 함.
- [ ] 워크플로우에서 사이드 팝업이 열렸을 때 EXC 를 이용하여 닫을 수 있도록 만들어야 함
- [ ] 전체 저장 기능이 있는데, Apply, Cancel 이 있다는 것이 사용자 경험에 좋지 않음. 전체 Save 버튼을 통해서 워크플로우를 저장할 수 있도록 해야 함.
